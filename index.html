California Academic Performance Index Data

The Academic Performance Index (API) is a measurement of academic performance and progress of individual schools in California.

A school's score or placement on the API is designed to be an indicator of a school's performance level and is calculated annually by the California Department of Education, primarily based on CST and CAHSEE tests.

Excerpt from Academic Performance Index (California public schools), Wikipedia

The list below contains CSV data files along with accompanying "column descriptions" JSON files which can help in programatically processing the CSVs.

The origin of this data is at http://www.cde.ca.gov/Ta/ac/ap/apidatafiles.asp. The original files were processed using the scripts available in this repository to make them easier to access and further process.
